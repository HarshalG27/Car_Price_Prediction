# 🚗 Car Price Prediction using Machine Learning

A GitHub-ready end-to-end machine learning project that predicts the resale price of used cars using a Streamlit web application.

## Features
- Car resale price prediction
- Trained Random Forest Regressor model
- Interactive Streamlit interface
- Ready for GitHub deployment

## Dataset
Based on the popular Kaggle Car Price Prediction dataset (Cardekho-style dataset).

## Project Structure
- `app.py` - Streamlit application
- `train_model.py` - Model training script
- `car_price_model.pkl` - Trained ML model
- `car_data.csv` - Dataset
- `requirements.txt` - Required Python libraries

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Model
- Algorithm: Random Forest Regressor
- Regression Task: Predict used car selling price
